# M1: 授業予定組立ツール 設計概要

## 概要

講師・カリキュラム・教室を管理し、動的計画法（DP）で時間割を自動生成するモジュール。教務管理者専用のバックエンドツールとして機能する。

M1は以下のサブモジュールで構成される。

## サブモジュール一覧

| サブモジュール | 説明 | 詳細 |
|--------------|------|------|
| [カリキュラム管理](curriculum.md) | 学科・講師・カリキュラム・タームの CRUD | データ入力・CSV インポート |
| [時間割自動生成](schedule-generation.md) | DP アルゴリズムによる自動配置・入れ替え | CSP・MRV・バックトラック |
| [施設予約](facility-booking.md) | 教室の予約管理（プラグインアーキテクチャ） | カレンダー連携付き |

## データフロー

```
カリキュラム管理 → 時間割自動生成 → 確定 → M2（データ統合）へエクスポート
                                          → 施設予約（教室使用状況に反映）
```

## アクセス制御

- カリキュラム管理・時間割生成: 教務管理者（admin）のみ
- 施設予約: 認証済みユーザー全般
- カリキュラム設定は editableUntil 期限内のみ編集可能
